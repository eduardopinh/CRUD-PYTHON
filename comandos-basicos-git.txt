Principais comandos básicos do Git

1) Configurando nome do usuário
git config --global user.name eduardopinh

2) Configurando o e-mail do usuário
git config --global user.email evictoreduardo09@gmail.com

3) Verificar o usuário
git config user.name

4) Verificar o e-mail
git config user.email

5) Iniciar um repositório
git init

6) Adicionar os arquivos
para o envio ao github
git add .

7) Adicionar mensagem
para o envio ao github

8)Alterar o nome da ramificação
do repositório local (computador
ou notebook) para "main"
git branch -M main

9) Referenciar o repositório local
para o repositório remoto (Github)
git remote add origin https://github.com/seu-repositorio-github.git

10) Realizar o envio do repositório
local para o remoto
git push -u origin main

OBS: Depois de confirmado o envio de repositório
local para o remoto, toda alteração posterior
deverá ser enviada ao Github com os comandos
a seguir
